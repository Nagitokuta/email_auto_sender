# セットアップガイド

## 1. Google スプレッドシートの作成

1. Google ドライブで新しいスプレッドシートを作成
2. 「電子メール自動送信管理」などの名前を付ける

## 2. シートの設定

### メール設定シート

- A1: テンプレート ID
- B1: 件名
- C1: 本文

### 宛先シート

- A1: No.
- B1: 名前
- C1: メールアドレス
- D1: ステータス
- E1: テンプレート ID

### 送信履歴シート

- A1: 日時
- B1: 宛先名
- C1: 宛先メールアドレス
- D1: 件名
- E1: 本文
- F1: テンプレート ID
- G1: ステータス

## 3. Google Apps Script の設定

1. スプレッドシートから「拡張機能」→「Apps Script」
2. 各.gs ファイルのコードを貼り付け
3. 保存して初回実行
